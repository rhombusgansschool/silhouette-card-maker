# DXF Cutting Templates (Borderless Variant)

Generated DXF cutting template files for borderless (3mm registration mark inset) layouts.

These files are generated by `generate_dxf.py` and are used as input for `dxf_to_studio3.py`
to create Silhouette Studio .studio3 files in the parent directory. Borderless variants use
a smaller registration mark inset to maximize the usable cutting area.

**Note:** DXF files are generated artifacts and are excluded from version control.
Only the .studio3 files in the parent directory are committed to the repository.
